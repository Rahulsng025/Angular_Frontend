# Angular_Frontend
This codebase was created to demonstrate a fully fledged application built with Angular that interacts with an actual backend server including get post operations, routing, pie-chart implimentation, and much more.
Additionally, there is an Angular 9 version of this codebase that you can fork and/or learn how to recreate.

* MongoDB : Document database – used by your back-end application to store its data as JSON (JavaScript Object Notation) documents
* Express (sometimes referred to as Express.js): Back-end web application framework running on top of Node.js
* Angular (formerly Angular.js): Front-end web app framework; runs your JavaScript code in the user's browser, allowing your application UI to be dynamic
* Node.js : JavaScript runtime environment – lets you implement your application back-end in JavaScript
